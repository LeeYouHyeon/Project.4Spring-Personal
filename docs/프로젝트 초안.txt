계정
- 관리자 개념 추가
	: 공지사항 작성
	: 모든 글과 댓글 삭제, 복구
	: 관리자는 아이디, 비밀번호, 이름 외의 정보 없음(프로필 사진은 게시판 아이콘 사용)
- 프로필 사진 추가(크기 제한)

0. 헤더
로고(메인 페이지로), 공지사항, 게시판, 로그인 / 아이디-로그아웃

1. 메인 페이지
- 사이트 소개글
- 인기글 : 최근 한 달 내에 추천수가 많은 글을 5개 추려 인기글 출력 형식대로 출력(사진이 없는 인기글은 썸네일에 게시판 로고를 띄움)
- 로그인한 경우 회원정보 페이지 링크, 즐겨찾기 페이지 링크, 개인정보 페이지 링크 출력
- 비회원 상태이면 로그인 버튼, 회원가입 버튼 출력

2. 회원가입 페이지
- 아이디 중복체크 기능(비동기)
- 정규식으로 비밀번호 조건 걸기

3. 로그인 페이지

4. 회원 정보 페이지
- 이름, email 확인 가능
- 회원의 게시글 및 댓글 모아보기(검색, 정렬 가능)
- 차단된 회원은 관리자만 열람 가능

5. 개인정보 페이지
- ID, 이름, email 확인 및 수정 가능
- 비밀번호 수정은 별도의 페이지로 처리

게시판 공통
- 글, 댓글의 작성자 부분에 ID 대신 이름을 출력
- 검색에 아이디와 이름 기준을 별도로 분리

7. 게시판 1 : 일반 게시판
- 비회원은 댓글만 작성 가능
- 검색이 적용되지 않은 1페이지의 최상단에 관리자가 지정한 공지사항 표기(페이징 기능과 별도로 지정)

8. 게시판 2. 공지사항 게시판(게시판 1과 별도의 jsp)

9. 일반 게시글 작성
- 작성자를 자신의 닉네임으로 표기

10. 공지사항 작성
- 최상단에 띄울지 말지 선택 가능. 띄우지 않음 / 상시 표기 / 기한 지정으로 선택지를 주고, 기한 지정을 고를 경우 date input을 제공

12. 게시글 상세 내용
- 관리자만 작성 가능
- 내용 확인은 일반 글과 같은 형식
- 공지사항은 댓글 작성 불가능


게시판 부가기능
1) 좋아요, 싫어요
- 회원당 게시글당 한 번씩 가능, 비회원은 불가능
- 취소 가능

2) 북마크
- 자주 볼 글(공지사항 포함)을 모아보는 기능
